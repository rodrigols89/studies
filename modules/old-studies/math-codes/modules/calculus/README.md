## Taxa de Varia√ß√£o

> **NOTE:**  
> A primeira coisa que voc√™ tem que ter em mente √© que **Taxa de Varia√ß√£o**, **Inclina√ß√£o da Fun√ß√£o** e **Derivada** *s√£o a mesma coisa*.

A **Taxa de Varia√ß√£o** indica:

 - A *rapidez (velocidade)* e a *dire√ß√£o* da mudan√ßa de uma *vari√°vel dependente (y)*.
 - Em rela√ß√£o √† varia√ß√£o da *vari√°vel independente (ùë•)*.

> **NOTE:**  
> Resumidamente, a *taxa de varia√ß√£o* indica quantos passos acontecem em `y (vari√°vel dependente)` quando eu dou um passo em `x (vari√°vel independente)`.

> **Olhando para o gr√°fico √© f√°cil, mas como identificar essa *mudan√ßa (Taxa de Varia√ß√£o)* para qualquer fun√ß√£o linear?**

 - A **diferen√ßa (Œîy)** entre **y<sub>1</sub>** e **y<sub>2</sub>**.
 - A **diferen√ßa (Œîx)** entre **x<sub>1</sub>** e **x<sub>2</sub>**.
 - **NOTE:** Por fim, n√≥s temos a raz√£o entre essas duas diferen√ßas.

![img](images/linear-rate-of-change-03.png)  
<!---

Agora, para resolver essa equa√ß√£o (formula) n√≥s vamos precisar de apenas de dois pares ordenados de valores **x** e **y**.

 - **Ap√≥s 1 segundo:**
   - *"x"* √© *"1"*.
   - *"y"* √© *"3"*.
 - **Ap√≥s 10 segundos:**
   - *"x"* √© *"10"*.
   - *"y"* √© *"21"*.

Logo:

![img](images/linear-rate-of-change-04.png)  
<!---
\\\mathbf{m = \frac{21 \ - \ 3}{10 \ - \ 1} = \frac{18}{9} = 2}

\\\mathbf{ m = 2}
--->

> **Mas o que significa esse 2**

 - Significa que a nossa *vari√°vel dependente (y)* est√° mudando (aumentando ou diminuindo) 2 em rela√ß√£o a *vari√°vel independente (x)*.
 - Ou seja, o ciclista est√° percorrendo `2 metros por segundo`.

> **NOTE:**  
> Outra observa√ß√£o √© que esse *2 (taxa de varia√ß√£o)* √© a *"derivada"* da nossa fun√ß√£o.

## Limite

Para entendermos sobre **"Limites"**, vamos imaginar que temos a seguinte fun√ß√£o:

![img](images/intro-to-limit-01.png)
<!---
\mathbf{f(x) = \frac{2x^{2} \ - \ x \ - 1}{x \ - \ 1}}
--->

De in√≠cio, vamos testar essa fun√ß√£o para `x = 0`:

![img](images/intro-to-limit-02.png)  
<!---
\\\mathbf{f(x) = \frac{2x^{2} \ - \ x \ - 1}{x \ - \ 1} = \frac{2 \times 0^{2} - 0 - 1}{0 - 1} = \frac{2 x 0 - 0 - 1}{-1} = \frac{0 - 0 - 1}{-1} = \frac{-1}{-1}}

\\\mathbf{f(x) = \frac{-1}{-1}} 
--->

Agora, vamos testar a fun√ß√£o para `x = 1`:

![img](images/intro-to-limit-03.png)  
<!---
\\\mathbf{f(x) = \frac{2x^{2} \ - \ x \ - \ 1}{x \ - \ 1} = \frac{2 \times 1^{2} - 1 - 1}{1 - 1} = \frac{2 \times 1 - 1 - 1}{0} = \frac{2 - 1 - 1}{0} = \frac{0}{0}}
\\
\\\mathbf{f(x) = \frac{0}{0}}
--->

> **Como assim?**  
> N√£o tem como dividir por zero!

 - **No contexto de limites, isso √© o que conhecemos como:**
   - *"Indetermina√ß√£o"*.
   - **NOTE:** Tamb√©m podemos dizer que *"1" est√° fora do dom√≠nio da minha fun√ß√£o* (neste exemplo).

## Como resolver limites com "Q.C.P" & "Diferen√ßa de dois quadrados"

De in√≠cio, imagine que n√≥s temos o seguinte limite:

![img](images/qcp-dots-01.png)  
<!---
\mathbf{\displaystyle \lim_{x \to 3}\frac{x^{2} - 9}{x - 3}}
--->

Como podemos ver no exemplo acima o limite √© `x = 3`:

![img](images/qcp-dots-02.png)  
<!---
\mathbf{\displaystyle \lim_{x \to 3}\frac{x^{2} - 9}{x - 3} = \frac{3^{2} - 9}{3 - 3} = \frac{9 - 9}{3 - 3} = \frac{0}{0}}
--->

> **NOTE:**  
> Vejam que fazia sentido o limite ser `x = 3`, pois, temos uma *"Indetermina√ß√£o"* nesse ponto.

Uma maneira de resolver esse tipo de problema √© sempre encontrar **"Quem Causa o Problema (Q.C.P)"**.

No contexto de limites *"Quem SEMPRE Causa o Problema"* vai ser:

 - O **"x"**;
 - E **seu oposto**.

> **Como assim?**

 - *Quem Causou o problema* n√£o foi `x = 3`?
 - Ent√£o, o oposto vai ser `x = -3`.

Por√©m, uma maneira mais simples de resolver isso √© a seguinte... Est√° vendo aquela parte que tem **"x"**, a **"setinha (‚Üí)"** e o **"3"**?

![img](images/qcp-dots-03.png)  

Ent√£o, agora voc√™ vai remover essa **setinha (‚Üí)** adicionar o sinal oposto do n√∫mero que vem depois da seta.

> **Como assim?**  
> Depois da seta (‚Üí) n√£o √© 3 (positivo)? Ent√£o, agora voc√™ vai adicionar o sinal de negativo (-) no lugar da seta.

```bash
x ‚Üí 3
|   |
|   |
(x - 3) <--- (Sinal oposto no lugar da seta)
```

> **Mas afinal o que √© esse (x - 3)?**  
> `(x - 3)` √© *Quem Causa o Problema (Q.C.P)* para essa equa√ß√£o (limite).

Ou seja, para resolver meu limite, primeiro precisamos remover `(x - 3)` da equa√ß√£o (limite).

> **Mas como?**

Bem, uma maneira √© fazer com que o nosso **Q.C.P (x - 3)** apare√ßa em cima e em baixo da equa√ß√£o (limite) para cancelar/eliminar um com o outro:

![img](images/qcp-dots-04.png)  

> **Mas como?**

Vamos voltar para o nosso limite para ver onde podemos manipular/fatorar para que seja poss√≠vel cancelar/eliminar `(x - 3)` da equa√ß√£o (limite):

![img](images/qcp-dots-01.png)  

Bem, n√≥s j√° temos `(x - 3)` no denominador.

> **Tem como eu fazer `(x - 3)` no numerador para cancelar/eliminar um com o outro?**

Como nossa equa√ß√£o (limite) √© quadr√°tica uma maneira seria utilizar a **"Diferen√ßa de dois Quadrados"** que permite reescrever express√µes quadr√°ticas na forma de um produto de dois bin√¥mios:

![img](images/qcp-dots-05.png)  
<!---
\mathbf{a^{2} - b^{2} = (a + b).(a - b)}
--->

> **Ok, mas como essa f√≥rmula funciona?**

Imagine que n√≥s queremos fatorar a express√£o `x¬≤ - 9` no numerador da nossa equa√ß√£o (limite) utilizando a **"Diferen√ßa de dois Quadrados"**:

![img](images/qcp-dots-06.png)
<!---
\mathbf{\displaystyle \lim_{x \to 3}\frac{x^{2} - 9}{x - 3}}
--->

Primeiro, n√≥s precisamos identificar quem √© o **"a"** e o **"b"** na express√£o:

```bash
a¬≤ = x¬≤
b¬≤ = 9
```

Agora se voc√™s prestem aten√ß√£o ver√£o que n√≥s temos 2 igualdades, `a¬≤ = x¬≤` e `b¬≤ = 9`. Ou seja, podemos simplicar aplicando opera√ß√µes iguais dos dois lados.

> **Como assim?**

Por exemplo, eu posso simplicar tirando a raiz quadrada dos dois lados das duas igualdades:

```bash
----------- ‚àö --------
|                    |
|                   \ /
a¬≤ = x¬≤             (a = x)
|                       / \
|                        |
----------- ‚àö ------------



----------- ‚àö --------
|                    |
|                   \ /
b¬≤ = 9              (b = 3)
|                       / \
|                        |
----------- ‚àö ------------
```

Depois de simplificarmos temos que `a = x` e `b = 3`. Agora vamos adicion√°-los na formula da **"Diferen√ßa de dois Quadrados"**:

![img](images/qcp-dots-07.png)  
<!---
\\\mathbf{a^{2} - b^{2} = (a + b).(a - b)}
\\
\\\mathbf{x - 3 = (x + 3).(x - 3)}
--->

> **Mas como isso pode me ajudar e resolver meu limite?**

Ent√£o:

 - O meu `x¬≤ - 9` n√£o foi simplificado para `x - 3`;
 - E `x - 3` n√£o √© igual a `(x + 3).(x - 3)`?

Logo, eu posos substituir no meu limite a express√£o `x¬≤ - 9` pela express√£o `(x + 3).(x - 3)`:

![img](images/qcp-dots-08.png)  
<!---
\mathbf{\displaystyle \lim_{x \to 3}\frac{x^{2} - 9}{x - 3} = \frac{(x + 3).(x - 3)}{x - 3} = x + 3}
--->

Vejam que n√≥s fizemos aparecer:

 - **Quem Causa o Problema (x - 3)** no numerador;
 - **Quem Causa o Problema (x - 3)** no denominador;
 - E por fim, cancelamos um pelo outro.

Viram que depois de cancelarmos **Quem Causa o Problema (x - 3)** sobrou apenas **x + 3**? Ent√£o, esse √© o limite da fun√ß√£o:

![img](images/qcp-05.png)  

> **NOTE:**  
> Ou seja, o limite para nossa fun√ß√£o √© 6.

Isso √© √∫til porque nem sempre teremos um gr√°fico para visualizar e determinar o limite da fun√ß√£o. Mesmo assim, apenas para este exemplo, vamos observar como seria em um gr√°fico:

![img](images/qcp-06.png)  

Vejam que:

 - **"3"** n√£o est√° no dom√≠nio dessa fun√ß√£o (bolinha est√° aberta/exclus√£o);
 - Quando `x` se aproxima (tende) a 3.
 - A nossa fun√ß√£o (resultado) se aproxima de 6.

> **NOTE:**  
> Ou seja, **"6"** √© o limite para essa fun√ß√£o.
