## `O que é um verbo?`

**Verbo** é a palavra que:

- **Expressa uma ação:**
  - João `correu` cinco quilômetros.
  - Maria `estudou` para o concurso.
  - Os alunos `resolveram` a prova.
  - O professor `explicou` a matéria.
  - Nós `viajaremos` nas férias.
- **Expressa um estado:**
  - A criança `está` feliz.
  - O céu `permanece` nublado.
  - Ela `continua` preocupada.
  - O paciente `anda` desanimado.
  - O café `ficou` frio.
- **Indica uma mudança de estado:**
  - A água `congelou`.
  - O leite `azedou`.
  - A fruta `amadureceu`.
  - O rapaz `envelheceu`.
  - A planta `cresceu`.
- **Indica um fenômeno da natureza:**
  - `Choveu` durante toda a noite.
  - `Nevou` na serra.
  - `Ventou` muito ontem.
  - `Trovejou` durante a tempestade.
  - `Anoiteceu` rapidamente.
- **Indica existência:**
  - `Há` muitos livros na biblioteca.
  - `Existe` uma solução para o problema.
  - `Existem` várias possibilidades.
  - `Havia` muitas pessoas na fila.
  - `Houve` um acidente na rodovia.
- **Indica ocorrência:**
  - `Aconteceu` um imprevisto.
  - `Ocorreu` um erro no sistema.
  - `Sucedeu` um fato inesperado.
  - `Sobreveio` uma crise econômica.
  - `Verificou-se` um aumento nas vendas.
- **Indica um processo:**
  - A criança `cresce` rapidamente.
  - A madeira `apodreceu` com a umidade.
  - O ferro `enferruja` facilmente.
  - O gelo `derreteu` ao sol.
  - A massa `fermentou` durante a noite.
- **Localiza esses fatos no tempo:**
  - Ontem eu `estudei`. *(passado)*
  - Hoje eu `estudo`. *(presente)*
  - Amanhã eu `estudarei`. *(futuro)*
  - Na semana passada nós `viajamos`. *(passado)*
  - No próximo mês nós `viajaremos`. *(futuro)*

## `Como identificar um verbo?`

Para identificar um verbo, verifique se a palavra:

- Indica uma ação.
- Indica um estado.
- Indica uma mudança de estado.
- Indica um fenômeno da natureza.
- Indica existência ou ocorrência.
- Pode ser conjugada.

### `Exemplos`

| Frase | O que expressa | Justificativa |
|--------|----------------|---------------|
| João `estuda` todos os dias. | Ação | Indica uma ação praticada pelo sujeito. |
| Os atletas `correram` a maratona. | Ação | Expressa uma ação realizada pelos atletas. |
| A professora `explicou` a matéria. | Ação | Indica uma ação executada pelo sujeito. |
| O engenheiro `construiu` a ponte. | Ação | Representa uma ação concreta. |
| Nós `resolveremos` o problema. | Ação | Expressa uma ação que ocorrerá no futuro. |
| Maria `está` feliz. | Estado | Indica o estado momentâneo do sujeito. |
| O menino `permanece` calado. | Estado | Expressa permanência em determinado estado. |
| Ela `anda` preocupada. | Estado | O verbo indica estado, e não deslocamento. |
| O diretor `continua` ausente. | Estado | Indica continuidade de um estado. |
| O paciente `vive` cansado. | Estado | O verbo "viver" equivale a "estar frequentemente". |
| A criança `cresceu` muito. | Mudança de estado | Indica transformação física ao longo do tempo. |
| A fruta `amadureceu`. | Mudança de estado | Expressa alteração de condição. |
| O leite `azedou`. | Mudança de estado | Indica mudança de qualidade. |
| A roupa `secou`. | Mudança de estado | Expressa alteração de estado físico. |
| A água `congelou`. | Mudança de estado | Indica transformação física. |
| `Choveu` durante a madrugada. | Fenômeno da natureza | Indica um fenômeno natural. |
| `Nevou` na serra. | Fenômeno da natureza | Expressa fenômeno meteorológico. |
| `Ventou` bastante ontem. | Fenômeno da natureza | Refere-se à ação do vento. |
| `Trovejou` a noite inteira. | Fenômeno da natureza | Expressa fenômeno atmosférico. |
| `Anoiteceu` rapidamente. | Fenômeno da natureza | Indica a chegada da noite. |
| `Há` muitos livros na biblioteca. | Existência | O verbo **haver** significa existir. |
| `Havia` poucas vagas disponíveis. | Existência | Indica existência no passado. |
| `Existe` uma solução para isso. | Existência | Indica que algo existe. |
| `Existem` diversas alternativas. | Existência | Expressa existência de vários elementos. |
| `Restam` apenas duas vagas. | Existência | Indica o que ainda existe ou permanece. |
| `Houve` um acidente na rodovia. | Ocorrência | O verbo indica que um fato aconteceu. |
| `Ocorreu` um erro no sistema. | Ocorrência | Expressa acontecimento. |
| `Aconteceu` um imprevisto. | Ocorrência | Indica que um fato ocorreu. |
| `Sucedeu` um fato curioso. | Ocorrência | Expressa ocorrência de um evento. |
| `Verificou-se` um aumento nas vendas. | Ocorrência | Indica que um fato foi constatado. |
| A massa `fermentou`. | Processo | Expressa um processo gradual. |
| O ferro `enferrujou`. | Processo | Indica um processo químico. |
| A madeira `apodreceu`. | Processo | Expressa deterioração gradual. |
| O gelo `derreteu`. | Processo | Indica transformação física. |
| A tinta `secou`. | Processo | Expressa um processo natural. |
| A criança `dorme` cedo. | Estado | O verbo indica o estado em que o sujeito se encontra. |
| O aluno `ficou` nervoso. | Mudança de estado | O verbo "ficar" indica alteração de estado. |
| A porta `permaneceu` fechada. | Estado | Indica manutenção de um estado. |
| O café `esfriou`. | Mudança de estado | Expressa transformação de temperatura. |
| O bebê `nasceu` saudável. | Processo | O nascimento é entendido como um processo biológico. |
| O rio `transbordou`. | Fenômeno da natureza | Refere-se a um fenômeno natural. |
| `Relampejou` durante a tempestade. | Fenômeno da natureza | Indica ocorrência de relâmpagos. |
| `Bastava` um telefonema. | Existência | O verbo indica suficiência/existência de algo necessário. |
| `Faltam` dez minutos para a prova. | Existência | Indica o que ainda resta para completar um período. |
| `Sobrou` comida após a festa. | Existência | Expressa o que permaneceu existente. |
| `Sobreveio` uma crise econômica. | Ocorrência | Indica um acontecimento inesperado. |
| O paciente `adoeceu`. | Mudança de estado | Passou do estado saudável para doente. |
| As folhas `caíram` no outono. | Processo | Indica um processo natural da vegetação. |
| O vinho `envelheceu` na adega. | Processo | Expressa um processo gradual ao longo do tempo. |
| O céu `escureceu` rapidamente. | Mudança de estado | Houve alteração do estado do céu. |

### `Dica`

Uma boa forma de reconhecer um verbo é verificar se ele pode ser conjugado, por exemplo:

Exemplos:

| Infinitivo | Presente | Pretérito | Futuro |
|------------|----------|-----------|---------|
| estudar | estudo | estudei | estudarei |
| correr | corro | corri | correrei |
| escrever | escrevo | escrevi | escreverei |

> **Atenção:**  
> Nem toda palavra terminada em **-ção**, **-mento** ou **-agem** é verbo.

Compare:

- Estudar → **Verbo**.
- Estudo → **Substantivo**.
- Construir → **Verbo**.
- Construção → **Substantivo**.
- Julgar → **Verbo**.
- Julgamento → **Substantivo**.
