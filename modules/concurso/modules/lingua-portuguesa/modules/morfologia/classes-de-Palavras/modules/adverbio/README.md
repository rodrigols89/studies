## `O que é um advérbio?`

**Advérbio** é a palavra `invariável` que:

- **Modifica um verbo.**
  - João correu `rapidamente`. → `rapidamente` modifica o verbo **correu**, indicando **como** João correu.
  - Maria chegou `cedo`. → `cedo` modifica o verbo **chegou**, indicando **quando** ela chegou.
  - Pedro falou `baixinho`. → `baixinho` modifica o verbo **falou**, indicando **como** Pedro falou.
  - O aluno respondeu `corretamente`. → `corretamente` modifica o verbo **respondeu**, indicando **como** respondeu.
  - Ela saiu `ontem`. → `ontem` modifica o verbo **saiu**, indicando **quando** ela saiu.
- **Modifica um adjetivo.**
  - O exercício está `muito` difícil. → `muito` intensifica o adjetivo **difícil**.
  - A prova foi `extremamente` fácil. → `extremamente` intensifica o adjetivo **fácil**.
  - Ele é `bastante` inteligente. → `bastante` intensifica o adjetivo **inteligente**.
  - A sala ficou `pouco` iluminada. → `pouco` modifica o adjetivo **iluminada**, reduzindo sua intensidade.
  - O café estava `demasiadamente` quente. → `demasiadamente` intensifica o adjetivo **quente**.
- **Modifica outro advérbio.**
  - Ela chegou `muito` cedo. → `muito` intensifica o advérbio **cedo**.
  - O atleta correu `bem` demais. → `demais` intensifica o advérbio **bem**.
  - O professor falou `muito` claramente. → `muito` intensifica o advérbio **claramente**.
  - O carro passou rápido `demais`. → `demais` intensifica o advérbio **rápido**.
  - O aluno respondeu `quase` corretamente. → `quase` modifica o advérbio **corretamente**, indicando aproximação.
- **Modifica toda a oração.**
  - `Infelizmente`, perdemos o voo. → `infelizmente` expressa a opinião do falante sobre toda a oração.
  - `Felizmente`, todos foram aprovados. → `felizmente` avalia positivamente toda a oração.
  - `Certamente`, ele voltará. → `certamente` demonstra certeza sobre toda a oração.
  - `Provavelmente`, choverá amanhã. → `provavelmente` expressa dúvida sobre toda a oração.
  - `Sinceramente`, não concordo. → `sinceramente` indica a atitude do falante em relação à oração inteira.
- **Indica tempo.**
  - Cheguei `ontem`. → `ontem` informa **quando** ocorreu a ação.
  - Viajarei `amanhã`. → `amanhã` informa **quando** ocorrerá a ação.
  - Estudo `sempre`. → `sempre` indica frequência da ação.
  - Ela chegou `cedo`. → `cedo` informa o momento da chegada.
  - Responderemos `depois`. → `depois` indica o momento da ação.
- **Indica lugar.**
  - O livro está `aqui`. → `aqui` indica o lugar onde o livro está.
  - Eles moram `longe`. → `longe` indica o lugar em relação ao falante.
  - A professora está `ali`. → `ali` indica localização.
  - Venha `cá`. → `cá` indica direção em relação ao falante.
  - O cachorro ficou `fora`. → `fora` indica o lugar.
- **Indica modo.**
  - Ela respondeu `bem`. → `bem` indica a maneira como respondeu.
  - O menino falou `calmamente`. → `calmamente` indica o modo da fala.
  - O atleta correu `rapidamente`. → `rapidamente` indica a maneira da ação.
  - O bebê dormiu `tranquilamente`. → `tranquilamente` indica o modo como dormiu.
  - O aluno escreveu `cuidadosamente`. → `cuidadosamente` indica a maneira como escreveu.
- **Indica intensidade.**
  - Estou `muito` cansado. → `muito` intensifica o estado de cansaço.
  - Ela é `bastante` dedicada. → `bastante` intensifica o adjetivo **dedicada**.
  - O café está `demasiadamente` quente. → `demasiadamente` aumenta a intensidade do adjetivo **quente**.
  - A sala ficou `pouco` iluminada. → `pouco` reduz a intensidade do adjetivo **iluminada**.
  - O exercício é `extremamente` difícil. → `extremamente` intensifica o adjetivo **difícil**.
- **Indica negação.**
  - `Não` irei à reunião. → `não` nega a ação verbal.
  - Ela `jamais` mentiu. → `jamais` reforça a negação da ação.
  - `Nunca` visitei esse lugar. → `nunca` indica que a ação não ocorreu em nenhum momento.
  - Ele `tampouco` respondeu. → `tampouco` reforça a ideia de negação.
  - `Nem` todos compareceram. → `nem` indica exclusão dentro de uma ideia negativa.
- **Indica afirmação.**
  - `Sim`, aceito a proposta. → `sim` confirma a informação.
  - `Certamente` voltarei. → `certamente` expressa certeza.
  - `Realmente` aconteceu. → `realmente` reforça a veracidade do fato.
  - `Efetivamente` houve mudanças. → `efetivamente` confirma a ocorrência.
  - `Decerto` ele vencerá. → `decerto` indica convicção.
- **Indica dúvida.**
  - `Talvez` ele venha. → `talvez` indica possibilidade.
  - `Possivelmente` choverá. → `possivelmente` expressa incerteza.
  - `Provavelmente` passaremos. → `provavelmente` indica probabilidade.
  - `Quiçá` consigamos. → `quiçá` expressa dúvida.
  - `Acaso` ele saiba a resposta. → `acaso` indica hipótese.
- **Indica inclusão.**
  - `Também` participei da reunião. → `também` inclui o falante.
  - `Até` João compareceu. → `até` inclui um elemento inesperado.
  - Ela `inclusive` foi premiada. → `inclusive` acrescenta informação.
  - `Ainda` estudamos juntos. → `ainda` acrescenta continuidade da informação.
  - `Mesmo` os iniciantes conseguiram. → `mesmo` inclui um grupo inesperado.
- **Indica exclusão.**
  - `Só` Maria respondeu. → `só` restringe a ação a Maria.
  - `Somente` os aprovados entrarão. → `somente` limita o grupo.
  - `Apenas` uma questão faltou. → `apenas` restringe a quantidade.
  - `Exclusivamente` servidores participaram. → `exclusivamente` limita os participantes.
  - `Unicamente` ele permaneceu. → `unicamente` restringe a ação a uma pessoa.
- **Indica ordem.**
  - `Primeiramente`, leia o enunciado. → `primeiramente` indica a primeira etapa.
  - `Depois`, resolva a questão. → `depois` indica sequência.
  - `Posteriormente`, confira a resposta. → `posteriormente` indica etapa seguinte.
  - `Finalmente`, entregue a prova. → `finalmente` indica a última etapa.
  - `Inicialmente`, organize o material. → `inicialmente` marca o começo da sequência.
- **Indica designação.**
  - `Nomeadamente`, participaram João e Maria. → `nomeadamente` especifica os elementos mencionados.
  - Compareceram `especialmente` os convidados. → `especialmente` destaca um grupo específico.
  - Foram chamados `particularmente` os finalistas. → `particularmente` especifica quem foi destacado.
  - `Especificamente`, trataremos da sintaxe. → `especificamente` delimita o assunto.
  - `Notadamente`, a gramática foi o tema mais cobrado. → `notadamente` destaca um elemento específico.

## `Como identificar um advérbio?`

Para identificar um advérbio, verifique se a palavra:

- Modifica um verbo.
- Modifica um adjetivo.
- Modifica outro advérbio.
- Expressa uma circunstância.
- É invariável (não varia em gênero nem em número).

### `Exemplos`

| Frase | Advérbio | O que modifica | Circunstância |
|--------|----------|----------------|---------------|
| João chegou **ontem**. | ontem | verbo | Tempo |
| Maria mora **aqui**. | aqui | verbo | Lugar |
| Pedro corre **rapidamente**. | rapidamente | verbo | Modo |
| Ana é **muito** inteligente. | muito | adjetivo | Intensidade |
| João corre **muito** rapidamente. | muito | advérbio | Intensidade |
| **Talvez** ele venha. | talvez | oração | Dúvida |

### `Dica`

Uma boa forma de reconhecer um advérbio é verificar se ele responde a perguntas como:

- **Quando?**
  - Ontem.
  - Hoje.
- **Onde?**
  - Aqui.
  - Ali.
- **Como?**
  - Bem.
  - Rapidamente.
- **Quanto?**
  - Muito.
  - Pouco.
- **É certo?**
  - Sim.
  - Certamente.
- **É negativo?**
  - Não.
  - Jamais.
- **Há dúvida?**
  - Talvez.
  - Provavelmente.

Outra característica importante é que o advérbio costuma permanecer igual, independentemente do gênero ou do número.

Exemplos:

- O menino corre **rapidamente**.
- A menina corre **rapidamente**.
- Os meninos correm **rapidamente**.
- As meninas correm **rapidamente**.

O advérbio **rapidamente** não sofre alteração.

> **Atenção:** Muitas palavras terminadas em **-mente** são advérbios de modo, mas **nem todo advérbio termina em -mente**.

Compare:

- Rapidamente → Advérbio.
- Lentamente → Advérbio.
- Hoje → Advérbio.
- Aqui → Advérbio.
- Nunca → Advérbio.
- Muito → Advérbio (quando modifica verbo, adjetivo ou outro advérbio).
