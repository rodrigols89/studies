## `O que é um adjetivo?`

O **adjetivo** é a palavra que `acompanha o substantivo` para:

> Substantivo em negrito: `**substantivo**`.
> Adjetivo em caractere especial: ``.

- **Caracterizá-lo.**
  - **homem** `alto`
  - **casa** `grande`
  - **carro** `veloz`
  - **árvore** `frondosa`
  - **cidade** `movimentada`
- **Qualificá-lo.**
  - **aluno** `inteligente`
  - **profissional** `competente`
  - **atleta** `dedicado`
  - **professora** `paciente`
  - **médico** `experiente`
- **Atribuir-lhe uma qualidade.**
  - **água** `limpa`
  - **tecido** `macio`
  - **fruta** `doce`
  - **livro** `interessante`
  - **música** `agradável`
- **Indicar seu estado.**
  - **porta** `aberta`
  - **janela** `fechada`
  - **criança** `doente`
  - **paciente** `recuperado`
  - **roupa** `molhada`
- **Indicar seu aspecto.**
  - **céu** `azulado`
  - **parede** `esverdeada`
  - **rosto** `avermelhado`
  - **cabelo** `grisalho`
  - **folha** `amarelada`
- **Indicar sua origem.**
  - **vinho** `francês`
  - **café** `brasileiro`
  - **queijo** `suíço`
  - **carro** `japonês`
  - **azeite** `português`
- **Indicar sua condição.**
  - **paciente** `enfermo`
  - **trabalhador** `desempregado`
  - **homem** `livre`
  - **aluno** `aprovado`
  - **cidadão** `aposentado`

## `Tipos de adjetivos`

- **Primitivo:** `não deriva de outra palavra da língua portuguesa.`
  - bom
  - mau
  - triste
  - azul
  - verde
- **Derivado:** `deriva de outra palavra.`
  - amoroso (amor)
  - pedregoso (pedra)
  - arenoso (areia)
  - estudantil (estudante)
  - famoso (fama)
- **Simples:** `possui apenas um radical.`
  - feliz
  - grande
  - alto
  - pobre
  - forte
- **Composto:** `possui dois ou mais radicais.`
  - luso-brasileiro
  - azul-escuro
  - econômico-financeiro
  - afro-brasileiro
  - verde-claro
- **Pátrio (gentílico):** `indica a origem, nacionalidade ou procedência.`
  - brasileiro
  - argentino
  - português
  - paraibano
  - europeu

## `Como identificar um adjetivo?`

Para identificar um adjetivo, verifique se a palavra:

- Caracteriza um substantivo.
- Atribui uma qualidade, estado, aspecto, origem ou condição ao substantivo.
- Concorda com o substantivo em gênero e número (quando variável).
- Pode ser intensificada por advérbios como **muito**, **bastante**, **tão**, **extremamente**, etc.

### `Exemplos`

| Expressão | Adjetivo | Substantivo |
|-----------|----------|-------------|
| Menino inteligente | inteligente | menino |
| Casa grande | grande | casa |
| Flor bonita | bonita | flor |
| Aluno dedicado | dedicado | aluno |
| Céu azul | azul | céu |
| Cidade brasileira | brasileira | cidade |

### `Dica`

Uma boa forma de reconhecer um adjetivo é verificar se ele responde à pergunta:

> **Como é o substantivo?**

Exemplos:

- **Como é o menino?**
  - Inteligente.
- **Como é a casa?**
  - Grande.
- **Como é a flor?**
  - Bonita.
- **Como é o aluno?**
  - Dedicado.

Outra dica é observar se o adjetivo pode ser intensificado.

Exemplos:

- Muito bonito.
- Extremamente feliz.
- Bastante difícil.
- Tão rápido.
- Pouco experiente.
