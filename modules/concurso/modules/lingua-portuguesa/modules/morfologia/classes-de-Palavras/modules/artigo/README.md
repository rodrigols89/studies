## `O que é um artigo?`

**Artigo** é a palavra que **acompanha o substantivo** para:

 - **Determiná-lo (Definidos)**
   - `o` carro
   - `a` casa
   - `os` alunos
   - `as` flores
 - **Ou indeterminá-lo (Indefinidos)**
   - `um` carro
   - `uma` casa
   - `uns` alunos
   - `umas` flores

Indicando também seu:

 - **Gênero:**
   - *Masculino:*
     - o carro
     - um livro
     - os meninos
     - uns professores
     - o computador
   - *Feminino:*
     - a casa
     - uma mesa
     - as meninas
     - umas professoras
     - a caneta
 - **Número:**
   - *Singular:*
     - o carro
     - a casa
     - um livro
     - uma flor
     - o aluno
   - *Plural:*
     - os carros
     - as casas
     - uns livros
     - umas flores
     - os alunos

## `Como identificar um artigo?`

Para identificar um artigo, verifique se a palavra:

- Acompanha um substantivo.
- Determina ou indetermina esse substantivo.
- Concorda com o substantivo em gênero e número.

### `Exemplos`

| Frase                     | Artigo | Substantivo |
|---------------------------|--------|-------------|
| **O** cachorro latiu.     | O      | cachorro    |
| **A** menina estudou.     | A      | menina      |
| **Um** aluno chegou.      | Um     | aluno       |
| **Uma** casa foi vendida. | Uma    | casa        |
| **Os** livros sumiram.    | Os     | livros      |
| **As** flores murcharam.  | As     | flores      |

### `Dica`

Se a palavra puder ser substituída por outro artigo sem deixar de acompanhar o substantivo, ela provavelmente é um artigo.

Exemplos:

- **O** cachorro → **Um** cachorro
- **A** professora → **Uma** professora
- **Os** alunos → **Uns** alunos
- **As** flores → **Umas** flores
