## `O que é um artigo?`

**Artigo** é a palavra que **acompanha o substantivo** para:

 - **Determiná-lo (Definidos)**
   - `o` carro
   - `a` casa
   - `os` alunos
   - `as` flores
 - **Ou indeterminá-lo (Indefinidos)**
   - `um` carro
   - `uma` casa
   - `uns` alunos
   - `umas` flores

Indicando também seu:

 - **Gênero:**
   - *Masculino:*
     - o carro
     - um livro
     - os meninos
     - uns professores
     - o computador
   - *Feminino:*
     - a casa
     - uma mesa
     - as meninas
     - umas professoras
     - a caneta
 - **Número:**
   - *Singular:*
     - o carro
     - a casa
     - um livro
     - uma flor
     - o aluno
   - *Plural:*
     - os carros
     - as casas
     - uns livros
     - umas flores
     - os alunos

## `Como identificar um artigo?`

Para identificar um artigo, verifique se a palavra:

- Acompanha um substantivo.
- Determina ou indetermina esse substantivo.
- Concorda com o substantivo em gênero e número.

### `Exemplos`

| Frase                     | Artigo | Substantivo |
|---------------------------|--------|-------------|
| **O** cachorro latiu.     | O      | cachorro    |
| **A** menina estudou.     | A      | menina      |
| **Um** aluno chegou.      | Um     | aluno       |
| **Uma** casa foi vendida. | Uma    | casa        |
| **Os** livros sumiram.    | Os     | livros      |
| **As** flores murcharam.  | As     | flores      |

### `Dica`

Se a palavra puder ser substituída por outro artigo sem deixar de acompanhar o substantivo, ela provavelmente é um artigo.

Exemplos:

- **O** cachorro → **Um** cachorro
- **A** professora → **Uma** professora
- **Os** alunos → **Uns** alunos
- **As** flores → **Umas** flores

## `Contrações dos artigos (Preposição + Artigo)`

> A **contração** acontece quando uma **preposição** encontra um **artigo** e eles passam a ser escritos como **uma única palavra**.

### `Quando isso acontece?`

Sempre que uma preposição vem imediatamente antes de um artigo.

Por exemplo:

```
Gosto de o livro.
```

Essa forma não é usada na língua padrão.

Ela sofre contração:

```
Gosto do livro.
```

### `Principais contrações cobradas em concursos`

**Preposição "de":**

| Forma separada | Contração |
| -------------- | --------- |
| de + o         | **do**    |
| de + a         | **da**    |
| de + os        | **dos**   |
| de + as        | **das**   |

**Preposição "em":**

| Forma separada | Contração |
| -------------- | --------- |
| em + o         | **no**    |
| em + a         | **na**    |
| em + os        | **nos**   |
| em + as        | **nas**   |

**Preposição a:**

| Forma separada | Contração |
| -------------- | --------- |
| a + o          | **ao**    |
| a + os         | **aos**   |

**Atenção:**  
Quando a preposição **a** encontra os artigos **a** ou **as**, ocorre **crase**:

```
a + a = à
a + as = às
```

Exemplo:

```
Fui à escola.
```

↓

```
à = a (preposição) + a (artigo)
```

### `Como identificar em uma prova?`

Sempre faça esta pergunta:

> **Existe um substantivo logo depois?**

Se existir, provavelmente a segunda parte da contração é um **artigo**.

```
do professor
```

Existe o substantivo **professor**.

Então:

```
do
↓
de + o
```

* **de** → preposição
* **o** → artigo

```
na escola
```

Existe o substantivo **escola**.

Então:

```
na
↓
em + a
```

## `O artigo sempre determina um substantivo, mesmo que haja outras palavras entre eles`

O importante é que essas palavras intermediárias façam parte do **mesmo sintagma nominal** (grupo de palavras cujo núcleo é um substantivo).

### `Exemplo 1`

> **...é o seu dom...**

```text
o      seu      dom
│       │        │
│       │        substantivo
│       pronome possessivo
artigo
```

O artigo **o** continua determinando **dom**, embora exista o pronome **seu** entre eles.

### `Exemplo 2`

> **Aquela era a minha resposta.**

```text
a      minha      resposta
│         │           │
│         │           substantivo
│         pronome possessivo
artigo
```

O artigo **a** determina **resposta**.

### `Exemplo 3`

> **Os dois alunos chegaram.**

```text
os      dois      alunos
│         │          │
│         │          substantivo
│         numeral
artigo
```

O artigo **os** determina **alunos**.

### `Exemplo 4`

> **A bela menina sorriu.**

```text
a      bela      menina
│        │          │
│        │          substantivo
│        adjetivo
artigo
```

O artigo **a** determina **menina**.

### `Exemplo 5`

> **Os três primeiros candidatos passaram.**

```text
os   três   primeiros   candidatos
│      │         │            │
│      │         │            substantivo
│      │         adjetivo
│      numeral
artigo
```

Mesmo distante, o artigo **os** continua determinando **candidatos**.

### `Regra geral`

O artigo **não precisa estar imediatamente antes do substantivo**.

Ele pode ser separado por palavras como:

* pronome possessivo;
* numeral;
* adjetivo;
* locuções adjetivas.

O que ele **não pode** é deixar de determinar um substantivo.

### `Dica para concursos`

Sempre que encontrar um possível artigo, faça a pergunta:

> **Qual substantivo este artigo está determinando?**

Se você conseguir responder, então realmente é um artigo.

Por exemplo:

* **o seu dom** → o artigo **o** determina **dom**.
* **as duas últimas provas** → o artigo **as** determina **provas**.
* **os melhores candidatos** → o artigo **os** determina **candidatos**.

Essa é uma técnica muito usada em análise morfológica e praticamente elimina as dúvidas entre **artigo** e **pronome**.
