## `O que é um substantivo?`

> **Substantivo** é a palavra que dá nome aos **seres**, **objetos**, **lugares**, **sentimentos**, **ações**, **estados**, **qualidades** e **conceitos**.

Por exemplo

- **Seres:**
  - Menino
  - Professora
  - Cachorro
  - Águia
  - Bactéria
- **Objetos:**
  - Mesa
  - Caderno
  - Computador
  - Relógio
  - Martelo
- **Lugares:**
  - Escola
  - Hospital
  - Praia
  - Biblioteca
  - Cidade
- **Sentimentos:**
  - Amor
  - Ódio
  - Alegria
  - Tristeza
  - Medo
- **Ações:**
  - Corrida
  - Estudo
  - Viagem
  - Leitura
  - Construção
- **Estados:**
  - Saúde
  - Doença
  - Repouso
  - Silêncio
  - Pobreza
- **Qualidades:**
  - Beleza
  - Honestidade
  - Inteligência
  - Coragem
  - Bondade
- **Conceitos:**
  - Liberdade
  - Justiça
  - Democracia
  - Igualdade
  - Verdade

## `Tipos de substantivos`

Os substantivos podem ser classificados de diversas maneiras.

- **Comum:** `nomeia seres de uma mesma espécie.`
  - cidade
  - cachorro
  - professor
  - livro
  - árvore
- **Próprio:** `nomeia um ser específico, distinguindo-o dos demais.`
  - João
  - Maria
  - Brasil
  - Recife
  - Amazonas
- **Concreto:** `nomeia seres que possuem existência própria, real ou imaginária.`
  - cadeira
  - cachorro
  - fada
  - Deus
  - anjo
- **Abstrato:** `nomeia sentimentos, ações, estados, qualidades ou conceitos, cuja existência depende de outro ser.`
  - amor
  - felicidade
  - estudo
  - coragem
  - beleza
- **Primitivo:** `não deriva de outra palavra da língua portuguesa.`
  - pedra
  - terra
  - flor
  - livro
  - ferro

- **Derivado:** `deriva de outra palavra.`
  - pedreira
  - terrestre
  - florista
  - livraria
  - ferreiro
- **Simples:** `possui apenas um radical.`
  - casa
  - flor
  - papel
  - sol
  - chuva
- **Composto:** `possui dois ou mais radicais.`
  - guarda-chuva
  - passatempo
  - girassol
  - segunda-feira
  - couve-flor
- **Coletivo:** `mesmo no singular, indica um conjunto de seres da mesma espécie.`

## `Como identificar um substantivo?`

Para identificar um substantivo, verifique se a palavra:

- Dá nome a um `ser`, `objeto`, `lugar`, `sentimento`, `ação`, `estado`, `qualidade` ou `conceito`.
- Pode ser acompanhada por um artigo.
- Pode receber um adjetivo.
- Pode variar em gênero e/ou número.

### `Exemplos`

| Palavra | É substantivo? | Motivo |
|----------|----------------|--------|
| menino | ✅ | Nomeia um ser. |
| mesa | ✅ | Nomeia um objeto. |
| escola | ✅ | Nomeia um lugar. |
| felicidade | ✅ | Nomeia um sentimento. |
| estudo | ✅ | Nomeia uma ação. |
| beleza | ✅ | Nomeia uma qualidade. |
| justiça | ✅ | Nomeia um conceito. |

### `Dica`

> Se `a palavra aceita naturalmente um artigo` e `está nomeando algo`, muito provavelmente é um substantivo.

Por exemplos:

- **O** menino.
- **A** mesa.
- **Uma** escola.
- **O** amor.
- **A** coragem.
- **A** justiça.

## `Plural em substantivos compostos`

Antes de formar o plural dos substantivos, descubra se cada elemento é:

* Substantivo
* Adjetivo
* Verbo
* Advérbio
* Preposição
* Numeral

Depois aplique a regra correspondente.

### `Substantivo + Substantivo`

➡️ Normalmente **os dois vão para o plural**.

| Singular           | Plural               |
| ------------------ | -------------------- |
| couve-flor         | couves-flores        |
| tique-taque        | tiques-taques        |
| peixe-boi          | peixes-bois          |
| cirurgião-dentista | cirurgiões-dentistas |
| carta-bilhete      | cartas-bilhetes      |

> **Macete:** dois substantivos → geralmente os dois variam.

### `Substantivo + Adjetivo`

➡️ Ambos vão para o plural.

| Singular        | Plural            |
| --------------- | ----------------- |
| amor-perfeito   | amores-perfeitos  |
| cachorro-quente | cachorros-quentes |
| guarda-noturno  | guardas-noturnos  |
| ponto-final     | pontos-finais     |
| obra-prima      | obras-primas      |

### `Verbo + Substantivo`

➡️ Apenas o substantivo varia.

> **O verbo permanece invariável.**

| Singular       | Plural          | Descrição                                                                                                                                                                                           |
| -------------- | --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| guarda-chuva   | guarda-chuvas   | **guarda** → `verbo` (*guardar*). Indica a ação de **guardar/proteger** da chuva. O verbo permanece invariável; apenas o substantivo **chuva** vai para o plural.                                   |
| guarda-roupa   | guarda-roupas   | **guarda** → `verbo` (*guardar*). Indica a ação de **guardar** roupas. O verbo permanece invariável; apenas o substantivo **roupa** vai para o plural.                                              |
| porta-bandeira | porta-bandeiras | **porta** → `verbo` (*portar*). Indica a ação de **portar/carregar** a bandeira. O verbo permanece invariável; apenas o substantivo **bandeira** vai para o plural.                                 |
| furta-cor      | furta-cores     | **furta** → `verbo` (*furtar*). Indica a ideia de **furtar/esconder** a cor, isto é, uma cor que muda conforme a luz. O verbo permanece invariável; apenas o substantivo **cor** vai para o plural. |
| beija-flor     | beija-flores    | **beija** → `verbo` (*beijar*). Indica a ação de **beijar** as flores ao sugar o néctar. O verbo permanece invariável; apenas o substantivo **flor** vai para o plural.                             |

> **Macete:** verbo nunca vai para o plural.

### `Advérbio + Substantivo ou Particípio`

➡️ Apenas o segundo elemento varia.

| Singular        | Plural                         | Descrição                                                                                                                                                                                                                                                                                                                 |
| --------------- | ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| abaixo-assinado | abaixo-assinados               | **abaixo** → `advérbio` de lugar. Modifica **assinado**, indicando que o documento foi assinado **abaixo** do texto. Como advérbios são invariáveis, apenas **assinado** vai para o plural.                                                                                                                               |
| alto-falante    | alto-falantes                  | **alto** → `advérbio` de intensidade. Modifica **falante**, indicando que o aparelho **fala alto**. O advérbio permanece invariável; apenas **falante** vai para o plural.                                                                                                                                                |
| sempre-viva     | sempre-vivas                   | **sempre** → `advérbio` de tempo. Modifica **viva**, indicando a ideia de **sempre viva**. Como advérbios não variam, apenas **viva** vai para o plural.                                                                                                                                                                  |
| recém-nascido   | recém-nascidos                 | **recém** → `advérbio` de tempo. Significa **há pouco tempo**, modificando **nascido**. O advérbio permanece invariável; apenas **nascido** vai para o plural.                                                                                                                                                            |
| mal-estar       | males-estar *(caso irregular)* | **mal** → neste caso **não é advérbio**, mas um `substantivo` (equivale a "o mal", "o sofrimento"). Por isso, a formação do plural foge à regra dos compostos com advérbio: **mal → males**, resultando em **males-estar**. Trata-se de um caso consagrado pelo uso e frequentemente tratado como exceção nas gramáticas. |

> **Macete:** advérbio não flexiona.

### `Palavra ligada por preposição`

➡️ Apenas a primeira palavra varia.

| Singular           | Plural              | Descrição                                                                                                                                                                                 |
| ------------------ | ------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| pé de moleque      | pés de moleque      | **de** → `preposição`. Liga os substantivos **pé** e **moleque**, formando uma locução substantiva. Em compostos ligados por preposição, apenas o primeiro substantivo vai para o plural. |
| mula sem cabeça    | mulas sem cabeça    | **sem** → `preposição`. Liga **mula** a **cabeça**, indicando ausência. Como há preposição entre os substantivos, apenas **mula** vai para o plural.                                      |
| pão de ló          | pães de ló          | **de** → `preposição`. Liga **pão** a **ló**, formando um nome composto. Apenas o primeiro substantivo (**pão**) vai para o plural.                                                       |
| fim de semana      | fins de semana      | **de** → `preposição`. Liga **fim** a **semana**, indicando relação de tempo. Apenas **fim** vai para o plural.                                                                           |
| folha de pagamento | folhas de pagamento | **de** → `preposição`. Liga **folha** a **pagamento**, indicando finalidade/especificação. Apenas **folha** vai para o plural.                                                            |

> **Macete:** apareceu uma preposição (*de, com, sem, por...*) → normalmente só o primeiro termo vai para o plural.

### `Palavras repetidas`

➡️ Geralmente ambos vão para o plural.

| Singular      | Plural          |
| ------------- | --------------- |
| tique-taque   | tiques-taques   |
| reco-reco     | recos-recos     |
| pingue-pongue | pingues-pongues |
| zigue-zague   | zigues-zagues   |
| corre-corre   | corres-corres   |
