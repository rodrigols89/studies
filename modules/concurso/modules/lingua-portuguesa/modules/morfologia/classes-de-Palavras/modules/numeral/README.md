## `O que é um numeral?`

**Numeral** é a palavra que `indica quantidade`, `ordem`, `multiplicação`, `fração` ou `conjunto numérico`, **relacionando-se, geralmente, a um substantivo**.

Por exemplo:

- **Quantidade (Cardinais):**
  - Um
  - Dois
  - Dez
  - Cem
  - Mil
- **Ordem (Ordinais):**
  - Primeiro
  - Segundo
  - Terceiro
  - Décimo
  - Centésimo
- **Multiplicação (Multiplicativos):**
  - Dobro
  - Triplo
  - Quádruplo
  - Quíntuplo
  - Sêxtuplo
- **Fração (Fracionários):**
  - Meio
  - Terço
  - Quarto
  - Quinto
  - Décimo
- **Conjunto numérico (Coletivos):**
  - Par
  - Dezena
  - Dúzia
  - Centena
  - Milhar

## `Como identificar um numeral?`

Para identificar um numeral, verifique se a palavra:

- Indica uma quantidade.
- Indica uma posição.
- Indica uma multiplicação.
- Indica uma fração.
- Indica um conjunto numérico exato.

### `Exemplos`

| Expressão | Numeral | Tipo |
|-----------|----------|------|
| Dois livros | dois | Cardinal |
| Terceiro colocado | terceiro | Ordinal |
| Dobro do salário | dobro | Multiplicativo |
| Meio copo | meio | Fracionário |
| Uma dúzia de ovos | dúzia | Coletivo |

### `Dica`

Uma boa forma de reconhecer um numeral é verificar se a palavra responde a uma das perguntas abaixo:

- **Quantos?**
  - Três alunos.
- **Em que posição?**
  - Segundo lugar.
- **Quantas vezes?**
  - O dobro.
- **Que fração?**
  - Um terço.
- **Qual conjunto numérico?**
  - Uma dúzia.

> **Atenção:**  
> Nem toda palavra que expressa quantidade é numeral.

Compare:

- **Dois** alunos. → **Numeral** (indica quantidade exata).
- **Muitos** alunos. → **Pronome indefinido** (quantidade imprecisa).
- **Alguns** livros. → **Pronome indefinido**.
- **Vários** carros. → **Pronome indefinido**.
