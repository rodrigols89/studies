## `O que é uma interjeição?`

> **Interjeição** é a palavra ou expressão **invariável** que transmite **emoções**, **sensações**, **sentimentos**, **estados de espírito**, **reações espontâneas** ou **apelos**, geralmente de forma isolada.

As interjeições podem expressar:

- **Alegria:**
  - Viva!
  - Oba!
  - Eba!
  - Uhu!
  - Legal!
- **Dor:**
  - Ai!
  - Ai de mim!
  - Ui!
  - Aiaiai!
  - Oh!
- **Surpresa:**
  - Nossa!
  - Uau!
  - Caramba!
  - Oh!
  - Meu Deus!
- **Admiração:**
  - Bravo!
  - Excelente!
  - Incrível!
  - Fantástico!
  - Magnífico!
- **Medo:**
  - Socorro!
  - Ajuda!
  - Meu Deus!
  - Credo!
  - Eita!
- **Silêncio:**
  - Psiu!
  - Silêncio!
  - Shiu!
  - Calma!
  - Atenção!
- **Saudação:**
  - Olá!
  - Oi!
  - Alô!
  - Bem-vindo!
  - Salve!
- **Desejo:**
  - Tomara!
  - Oxalá!
  - Deus queira!
  - Quem me dera!
  - Assim seja!
- **Alívio:**
  - Ufa!
  - Ainda bem!
  - Graças a Deus!
  - Finalmente!
  - Que alívio!
- **Chamamento:**
  - Ei!
  - Psiu!
  - Olá!
  - Ô!
  - Hei!

**📌 Observação importante para concursos:**  
Algumas interjeições podem pertencer a mais de uma categoria, dependendo do contexto. As bancas costumam explorar justamente essa variação.

| Interjeição   | Pode indicar                     |
| ------------- | -------------------------------- |
| **Oh!**       | Surpresa, admiração, dor         |
| **Meu Deus!** | Surpresa, medo, dor              |
| **Nossa!**    | Surpresa, admiração              |
| **Psiu!**     | Silêncio, chamamento             |
| **Olá!**      | Saudação, chamamento             |
| **Eita!**     | Surpresa, medo, admiração        |
| **Caramba!**  | Surpresa, indignação, admiração  |
| **Socorro!**  | Medo, desespero, pedido de ajuda |
| **Bravo!**    | Admiração, aprovação             |
| **Ufa!**      | Alívio, satisfação               |

> **NOTE:**  
> Dica para concursos: a interjeição não possui significado fixo. Seu valor semântico depende da entonação e do contexto. Por isso, a mesma palavra pode ser classificada de maneiras diferentes em questões de interpretação e gramática. Isso é especialmente cobrado por bancas como FGV, Cebraspe e FCC.

## `Como identificar uma interjeição?`

Para identificar uma interjeição, verifique se a palavra:

- Expressa uma emoção ou reação espontânea.
- Aparece geralmente de forma isolada.
- Não possui relação sintática com outros termos da oração.
- É invariável.
- Pode ser acompanhada de ponto de exclamação.

### `Exemplos`

| Frase | Interjeição | Sentimento expresso |
|--------|-------------|---------------------|
| **Ai!** Machuquei-me. | Ai | Dor |
| **Oba!** Vamos viajar. | Oba | Alegria |
| **Nossa!** Que surpresa. | Nossa | Espanto |
| **Ufa!** Acabou o trabalho. | Ufa | Alívio |
| **Psiu!** Venha aqui. | Psiu | Chamamento |

### `Dica`

Uma boa forma de reconhecer uma interjeição é perguntar:

> Essa palavra representa uma reação emocional ou uma manifestação espontânea?

Se a resposta for sim, provavelmente é uma interjeição.

Exemplos:

- **Eba!** Passei na prova.
  - Expressa alegria.

- **Ai!** Bati o joelho.
  - Expressa dor.

- **Nossa!** Que lugar bonito.
  - Expressa surpresa.

### `Atenção`

A mesma palavra pode pertencer a classes gramaticais diferentes dependendo do contexto.

Exemplos:

- **Nossa!** Que carro bonito.
  - Interjeição (expressa surpresa).

- **Nossa** casa é grande.
  - Pronome possessivo (indica posse).

Outro exemplo:

- **Bravo!** Você venceu.
  - Interjeição (expressa aprovação).

- O homem **bravo** saiu.
  - Adjetivo (caracteriza o substantivo homem).
