## `Onomatopeia`

**Onomatopeias** são palavras criadas para **imitar ou representar sons** produzidos por *pessoas*, *animais*, *objetos* ou *fenômenos da natureza*.

> Em outras palavras, a palavra tenta reproduzir, na escrita, o som que ouvimos.

Por exemplo:

| Som             | Onomatopeia        |
| --------------- | ------------------ |
| Gato            | **miau**           |
| Cachorro        | **au-au**          |
| Galo            | **cocoricó**       |
| Relógio         | **tic-tac**        |
| Campainha       | **trim-trim**      |
| Batida na porta | **toc-toc**        |
| Explosão        | **bum!**           |
| Tiro            | **pá!**, **bang!** |

### `Como identificar uma onomatopeia?`

Pergunte:

> **A palavra foi criada para imitar um som?**

* Se **sim**, provavelmente é uma **onomatopeia**.
* Se **não**, não é.

### `Por que "bem-te-vi" é uma onomatopeia?`

O nome da ave **bem-te-vi** surgiu porque o seu *canto* lembra a sequência de sons:

> **"bem-te-vi"**

Ou seja, as pessoas passaram a chamar o pássaro pelo som que ele parece emitir.

Assim:

* Som do pássaro → **"bem-te-vi"**
* Nome da ave → **bem-te-vi**

**NOTE:**  
Por isso, a palavra tem **origem onomatopaica**, pois nasceu da imitação do canto da ave.

### `Outros animais com nomes de origem onomatopaica`

| Animal          | Som que inspirou o nome |
| --------------- | ----------------------- |
| **Bem-te-vi**   | "bem-te-vi"             |
| **Cuco**        | "cuco"                  |
| **Quero-quero** | "quero-quero"           |
| **Uru**         | "uru"                   |
| **Jaó**         | "jaó"                   |

Esses nomes surgiram porque o canto do animal foi reproduzido na linguagem.

### `Atenção para concursos`

Nem toda palavra que representa um animal é uma onomatopeia.

Por exemplo:

* **cachorro** → ❌ não é onomatopeia.
* **gato** → ❌ não é onomatopeia.
* **bem-te-vi** → ✅ possui origem onomatopaica.
* **cuco** → ✅ possui origem onomatopaica.

Isso ocorre porque **cachorro** e **gato** não foram nomeados pela imitação de seus sons, enquanto **bem-te-vi** e **cuco** receberam seus nomes justamente por reproduzirem o canto característico dessas aves.
